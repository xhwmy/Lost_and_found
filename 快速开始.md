## 🚀 快速启动

### 方法一：使用启动脚本（推荐）
1. 双击 `start.bat` 文件
2. 等待依赖安装和服务启动
3. 浏览器访问 http://127.0.0.1:5000

### 方法二：手动启动
```bash
# 1. 进入项目目录
cd campus_crowdfunding

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行应用
python app.py
```

## 🔧 常见问题

### 1. 依赖安装失败
```bash
# 使用国内镜像源
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 2. 数据库错误
删除 `crowdfunding.db` 文件，重新运行 `python app.py` 自动创建新数据库

### 3. 端口被占用
修改 `app.py` 最后一行，更改端口号：
```python
app.run(host='0.0.0.0', port=5001, debug=True)  # 改为5001
```

### 4. 模板找不到
确保目录结构正确，`templates` 文件夹与 `app.py` 在同一目录
